Registration webApp

Registration webApp

This app was built using nodeJs and ExpressJs. It takes in a license plate number from the Western Cape and displays it. The App also allows you to filter the plates depending on its code location if a plate from that specific location has been added.

It also has the option to reset the database tables or dynamically delete plates.

Settings bill webApp

Settings bill webApp

This webApp is used to calculate mobile usage bills. It takes in the price of sms and calls and takes in a warning level and critical level. If the warning level is reached it changes the total to orange and if critical is reached it changes the total to red and stops incrementing

The app also records the all the values that has been added with a time stamp.
